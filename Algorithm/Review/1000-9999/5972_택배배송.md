# BOJ 5972 택배 배송

## 사고 흐름

전형적인 다익스트라 구현 문제였다.

입력되는 간선의 정보가 **양방향 정보**이므로 graph를 만들 때 반대 쪽도 연결해주어야 한다.

방문한 노드는 최단거리로 갱신되어 있으므로, 방문하지 않은 노드를 찾는 로직을 다음과 같이 처리해서 메모리를 줄였다.

``` java
// 현재 노드로 가는 비용(c.w)이 최단 거리(dist[c.e])가 아닐 때
if (dist[c.e] < c.w) continue; 
```
